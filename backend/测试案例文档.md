# 基于OpenHarmony的智慧农业控制系统

# 测试案例文档

---

## 1 测试概述

### 1.1 测试方法

本文档采用**黑盒测试**方法，主要运用**等价类划分法**和**边界值分析法**设计测试用例。等价类划分法将输入数据划分为若干等价类，从每个等价类中选取代表性数据进行测试，以较少的测试用例覆盖较多的测试场景。

### 1.2 测试环境

| 项目 | 配置 |
|------|------|
| 测试设备 | 华为Mate 60 Pro / HarmonyOS NEXT模拟器 |
| 操作系统 | OpenHarmony 6.0.2 (API 22) |
| 后端服务器 | Python 3.11 + FastAPI 0.104 |
| 数据库 | MySQL 8.0.35 |
| 网络环境 | 局域网WiFi (100Mbps) |
| 开发工具 | DevEco Studio 5.0 |

### 1.3 等价类划分说明

| 符号 | 含义 |
|------|------|
| 有效等价类 | 符合程序规格说明的合理输入数据集合 |
| 无效等价类 | 不符合程序规格说明的不合理输入数据集合 |
